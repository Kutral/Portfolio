## 🚀 Overview

This repository hosts a high-performance portfolio application. It leverages the latest frontend technologies to deliver a blazing-fast, responsive, and aesthetically pleasing interface. The focus is on clean architecture, beautiful micro-interactions, and a seamless narrative flow.

## 🎨 Features

- **Immersive Animations:** Powered by Framer Motion for smooth page transitions and element interactions.
- **Responsive Design:** Mobile-first approach ensuring a perfect layout across all devices using Tailwind CSS.
- **Type Safety:** Built entirely with TypeScript for robust, error-free code.
- **Blazing Fast:** Vite provides instantaneous hot module replacement (HMR) and optimized production builds.
- **Modern UI/UX:** Clean typography, thoughtful spacing, and engaging visual hierarchy.

## 🛠️ Tech Stack

- **Framework:** [React 18](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 💻 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (v18+ recommended).

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Navigate into the project directory:**
   ```bash
   cd portfolio
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

### Development

To start the local development server:

```bash
npm run dev
```
Open your browser and visit `http://127.0.0.1:5173/` (or the port specified by Vite) to view the application.

### Building for Production

To create an optimized production build:

```bash
npm run build
```
This command compiles the TypeScript files and bundles the assets into the `dist` directory.
